# 📔 Дневник погоды

Консольное приложение на C# для получения, сохранения и просмотра погодных данных. Использует бесплатные API Open-Meteo и поддерживает работу с часовыми поясами.

## ✨ Возможности

- 🌍 Определение координат и часового пояса по названию города
- 🌤️ Получение погоды на сегодня (по местному времени города)
- 📅 Получение погоды на любую прошедшую дату
- 💾 Сохранение записей в локальный JSON-дневник
- 📊 Просмотр дневника в виде таблицы с сортировкой по дате
- 📈 Простая статистика по городу (мин/макс температура, осадки, давление, ветер)
- 🕐 Корректная работа с часовыми поясами (IANA)
- 🎨 Цветовое логирование и эмодзи в выводе

## 🛠️ Используемые технологии

- .NET (C#)
- Open-Meteo Weather API
- Open-Meteo Geocoding API
- System.Text.Json + Source Generation
- Работа с часовыми поясами (TimeZoneInfo)
